# Ethereum and Crypto Terminology Reference

## Core Ethereum Terms

| Term | Correct Usage | Notes |
|------|--------------|-------|
| Ethereum | Always capitalized | Proper noun — the network |
| ether | Lowercase | The native currency; capitalize only at sentence start |
| ETH | Always capitalized | Ticker symbol for ether |
| Mainnet | Capitalized for Ethereum | "Claims are available only on Mainnet" |
| mainnet | Lowercase for other chains | "Arbitrum's mainnet has been paused" |
| Ethereum Mainnet | Both capitalized | Full proper reference to Ethereum's main chain |
| The Merge | Both capitalized | Proper noun — Ethereum's transition to PoS |
| proof-of-work (PoW) | Hyphenated, lowercase | Capitalize only at sentence start |
| proof-of-stake (PoS) | Hyphenated, lowercase | Capitalize only at sentence start |
| smart contract | Lowercase, two words | Capitalize only at sentence start |
| zero-knowledge | Hyphenated, lowercase | Capitalize only at sentence start |
| ZK-proof | "ZK" capitalized, hyphenated | Abbreviated form |
| ZK-rollup | "ZK" capitalized, hyphenated | Abbreviated form |
| onchain | One word, no hyphen | Not "on-chain" or "on chain" |
| offchain | One word, no hyphen | Not "off-chain" or "off chain" |
| DeFi | Capital D, capital F | Decentralized Finance |
| dapp | Lowercase | Decentralized application |
| gas | Lowercase | Transaction fee unit |
| wei | Lowercase | Smallest denomination of ether |
| gwei | Lowercase | Denomination of ether (10^9 wei) |

## Token Ticker Conventions

- Always use $ prefix for tickers: $ETH, $BTC, $CRV, $ARB, $UNI, $AAVE
- Tickers are always fully capitalized
- Use the ticker when referring to the token as a financial asset
- Use the protocol name when referring to the project/organization

```
GMI: Aave governance votes to deploy v4 on Avalanche; $AAVE surges 12% on the news
```

## Date Format

Use "D-Mon-YYYY" format: 2-Nov-2023, 15-Jan-2024

## Dash Usage

- **Hyphen (-)**: Compound words (proof-of-stake, zero-knowledge)
- **En dash (&ndash;)**: Ranges and connections (2020&ndash;2024, Ethereum&ndash;Bitcoin bridge)
- **Em dash (&mdash;)**: Strong breaks and emphasis within headlines

## American English

Use American English spelling exclusively:
- "decentralized" not "decentralised"
- "tokenized" not "tokenised"
- "analyze" not "analyse"
- "favor" not "favour"
